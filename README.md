Controls: 

    Movement:
    
        W / Arrow Key Up  - Move Up
        S / Arrow Key Down  - Move Down
        A / Arrow Key Left  - Move Left
        D / Arrow Key Right  - Move Right
        Space - Jump
    
    Combat:
        CTRL / Left Mouse Button - Attack
        Space + CTRL / Left Mouse Button - Slam Attack
